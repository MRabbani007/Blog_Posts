# Git & Github

## Version Control

## Life Cycle
1. Initiate Git: add git to project
```
git init
```
2. check status
```
git status
```
3. Stage: add files
```
git add .
```
4. Commit: commit current changes
``` 
git commit -m "message"
```
5. Push: Push to remote
```
git push
```
